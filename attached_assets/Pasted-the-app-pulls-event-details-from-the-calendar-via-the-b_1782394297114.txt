   - Checks if the packet is a UDP packet (IP protocol byte = 17)
   - Checks if destination port is 53
   - Returns true if both conditions met

   - Skips the IP header (first 20 bytes for IPv4)
   - Skips the UDP header (next 8 bytes)
   - Parses the DNS question section:
     - Reads length-prefixed labels until null byte

   - Takes the original DNS query packet
   - Returns a DNS response packet with RCODE = 3 (NXDOMAIN)
   - Copies the transaction ID from the original query
   - Sets QR bit to 1 (response)
   - This is what gets sent back to the app trying to resolve a blocked domain

Include full byte-level parsing with clear comments explaining each offset. Show complete file.
